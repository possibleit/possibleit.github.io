#python正则表达式

###python中的re模块，python正则表达式模块

  • 正则表达式的语法规则

    - 一般字符
    - .     匹配除换行符'\n'之外的任意字符
    - \     转义字符
    - [...] 字符集(字符类)。对应位置可以是字符集中任意字符
        -   示例 a[bcd]e 匹配 abe ace ade
    - 预定义字符集
    - \d    数字[0-9]
    - \D    非数字[^\d]
    - \s    空白字符[<空格>\t\r\n\f\v]
    - \S    非空白字符[^\s]
    - \w    单词字符[A-Za-z0-9]
    - \W    非单词字符[^\w]
    - 数量词(一般用在字符后边，表示作用次数)
    - *     匹配前一字符0或无限次
    - +     匹配前一字符1或无限次
    - ？    匹配前一字符0或1次
    - {m}   匹配前一字符m次
    - {m,n} 匹配前一字符m到n次
    - 边界限定
    - ^     匹配字符串开头(在多行模式中匹配每一行开头)
    - $     匹配字符串末尾(在多行模式中匹配每一行末尾)
    - \A    仅匹配字符串开头
    - \Z    仅匹配字符串末尾
    - 逻辑，分组
    - |     或，代表左右表达式任意匹配一个，一般先匹配左边表达式，一旦成功则跳过右边表达式
    - (...) 被包括起来的表达式将作为分组，从表达式左边开始,每遇到一个分组的左括号，
            编号+1。分组作为一个整体，可以后接数量词限定
    - (?P<name>) 
            分组，除了原编号外再指定一个别名
    - \<number>
            引用编号为<number>的分组匹配到的字符串
    - (?P=name)
            引用别名为<name>的分组匹配到的字符串

  • python的re模块

    • python自带的re模块，提供了对正则的支持，主要函数如下
    #返回pattern对象
re.compile(string[,flag])
#以下为匹配所用函数
re.match(pattern, string[,flags])
re.search(pattern, string[,flags])
re.split(pattern, string[,flags])
re.findall(pattern, string[,flags])
re.finditer(pattern, string[,flags])
re.sub(pattern, repl, string[,count])
re.subn(pattern, repl, string[,count])

  • 参数flag表示匹配模式，可使用&，| 等

    #可选值
re.I    忽略大小写
re.M    多行模式
re.S    点任意匹配符，改变‘.’的行为
re.X    详细模式，正则表达式可以是多行，忽略空白字符，并可以加入注释

  • re.match(pattern, string[,flags])

    #方法将会从字符串开头开始，尝试匹配pattern，一直向后匹配若匹配不成功
    则返回None,匹配成功则终止
import re
pattern = re.compile(r'hello')#r代表‘原生字符串’
result = re.match(pattern, 'hello world')
print(result.group())
#output hello
    • re,match返回一个match对象，包含了大多数这次匹配的信息，对象属性方法如下 
      属性
string:匹配时使用的文本
re：匹配时使用的pattern对象
pos：文本中正则表达式开始搜索的索引
endpos：正则表达式结束搜索的索引
lastindex：最后一个捕获的分组在文本中的索引，若果没有捕获分组，将为None
lastgroup：最后一个捕获的分组的别名，如果没有别名，将为None
方法
group([group1,..]):
     获得一个或多个分组截获的字符串；指定多个参数时将以元组的形式返回。
         编号0代表整个匹配的字符串
groups([default])：
     以元组形式返回全部分组截获的字符串
groupdict([default]):
     返回一有别名的组的别名为键、以该组截获字符串为值得字典。没有别名的组不包含在内
start([group]):
     返回指定的组截获的子串在string中的起始索引
end([group]):
     返回指定的组截获的子串在string中的结束索引(子串的最后一个字符的索引加一)
sapn([group]):
     返回(start(group),end(group))

  • re.search(pattern, string[,flags])

    search方法与match方法类似，区别在于match()只检测re是不是在string的开始位置匹配，
search()会扫描整个string匹配，match只有在0位置匹配成功的话才有返回值，否则返回None
同样，search返回的对象同样有match返回对象的属性和方法

  • re.split(pattern, string[,maxsplit])

    按照能够匹配的子串将string分割后返回列表，maxsplit用于指定最大分割次数，不指定将全部分割。

  • re.findall(pattern, string[,falgs])

    搜索string，以列表形式返回全部能匹配的子串

  • re.finditer(pattern, string[,flags])

    搜索string，返回一个能够顺序访问每一个匹配结果的迭代器

  • re.sub(pattern, repl, string[,count])

    使用repl替换string中每一个匹配的子串，返回替换后的字符串

  • re.subn(pattern, repl, string[,count]) 

    返回(sub(repl,string[,count]),替换次数)
